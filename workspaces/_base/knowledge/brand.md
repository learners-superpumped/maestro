# Brand Guidelines

## 브랜드 정체성
- **이름**: ClawOps
- **포지셔닝**: 한국 AI 개발자를 위한 전화·문자 통신 API 플랫폼
- **미션**: AI Agent가 전화하고 문자하는 세상을 만든다

## 브랜드 퍼소나
- 기술에 깊지만 설명은 쉽게
- 개발자 커뮤니티의 동료이자 선배
- 트렌드에 빠르게 반응하는 빌더

## 핵심 키워드
- AI Agent, 통신 API, 한국어 음성, 개발자 도구
- 자동화, 워크플로우, 빌더, 프로덕트

## 차별점
- 한국 통신 환경에 특화 (한국 번호, 한국어 TTS/STT)
- AI Agent 연동에 최적화된 SDK 및 MCP 지원
- SIP 서버 불필요, API 호출만으로 통화 가능
