# Technical SEO Checklist

Comprehensive checklist for SEO audits and optimization.

## Crawlability
- [ ] XML sitemap exists and is up to date
- [ ] Robots.txt is properly configured
- [ ] No unintentional noindex tags
- [ ] Canonical URLs are correctly set

## Performance
- [ ] Core Web Vitals within thresholds (LCP, FID, CLS)
- [ ] Images are optimized and use modern formats (WebP, AVIF)
- [ ] CSS and JS are minified and compressed
- [ ] Server response time < 200ms

## On-Page
- [ ] Unique title tags on all pages
- [ ] Meta descriptions on all key pages
- [ ] Proper heading hierarchy (single H1 per page)
- [ ] Alt text on all meaningful images
- [ ] Structured data (schema.org) where applicable

## Mobile
- [ ] Responsive design verified
- [ ] Mobile-friendly test passes
- [ ] Touch targets are appropriately sized
