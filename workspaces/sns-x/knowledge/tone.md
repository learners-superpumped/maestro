# X/Twitter Tone of Voice

Define the brand's tone of voice for X posts here.

## Guidelines
- Concise and punchy — every character counts (280 limit)
- Direct and confident
- Use hashtags purposefully, not excessively
- Thread format for longer narratives (number threads clearly)
- Emojis: use sparingly for emphasis, not decoration

## Thread Format
- Hook in the first tweet (grab attention in <100 chars)
- Number threads: 1/, 2/, 3/ etc.
- End threads with a clear CTA
